## Comments

### useComments

[Available options](https://developer.wordpress.org/rest-api/reference/comments/#list-comments)

Default query (latest 10 comments):

```jsx
const { data, loading, error } = useComments();
```

Comments meeting the criteria:

```jsx
const { data, loading, error } = useComments({
  /* WordPress REST API options */
});
```

Single comment (by id):

```jsx
const { data, loading, error } = useComments(/* comment id */);
```

Multiple comments (by id):

```jsx
const { data, loading, error } = useComments(/* array of comment ids */);
```
