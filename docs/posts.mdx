## Posts

### usePosts

[Available options](https://developer.wordpress.org/rest-api/reference/posts/#list-posts)

Default

```jsx
const { data, loading, error } = usePosts();
```

Single Post (with specified ID)

```jsx
const { data, loading, error } = usePosts(/* post id */);
```

Single Post

```jsx
const { data, loading, error } = usePosts({
  // options here
});
```
